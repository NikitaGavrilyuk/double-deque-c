# Double-Ended Queue (Deque) Implementation

This repository contains a C++ implementation of a Double-Ended Queue (Deque) data structure, developed as part of a course project.

## Features
* Insert elements at both the front and the rear of the queue.
* Delete elements from both ends.
* File-based data persistence via `DoubleDeque.dat`.

## How to Build and Run
You can compile and run the project using any standard C++ compiler (e.g., G++):

```bash
g++ main.cpp -o deque
./deque
