# Double-Ended Queue (Deque) Implementation

This repository contains a C implementation of a Double-Ended Queue (Deque) data structure, developed as a coursework project. 

*Note: The source code is written in pure C, but is saved with a `.cpp` extension to meet specific university environment requirements.*

## Features
* Insert elements at both the front and the rear of the queue.
* Delete elements from both ends.
* File-based data persistence via `DoubleDeque.dat`.

## How to Build and Run
Since the file uses a `.cpp` extension, it is recommended to compile it using a standard C++ compiler (e.g., G++) to avoid extension conflicts:

```bash
g++ main.cpp -o deque
./deque
